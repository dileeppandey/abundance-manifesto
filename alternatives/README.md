# Open Alternatives to Proprietary Systems

This directory is a curated map of open-source, cooperative, and commons-based alternatives to proprietary products and platforms.

It is not just a list. It is an argument.

Every entry here is evidence that the [Abundance Manifesto's](../MANIFESTO.md) five principles are not theoretical — they are already being practiced. Open tools exist. Cooperatives work. Distributed manufacturing is real. This directory maps the future that is already being built.

---

## How Entries Are Organized

Each alternative is tagged with the manifesto principle it most embodies:

- **[I] Radical Openness** — open source, open hardware, open data
- **[II] Design Global, Make Local** — distributed manufacturing, local production
- **[III] Regenerative Circularity** — repair-first, modular, zero-waste design
- **[IV] Cooperative Stewardship** — worker/user-owned platforms and services
- **[V] Access Over Ownership** — shared tools, libraries, service models

Many tools embody multiple principles — we tag the most prominent one.
