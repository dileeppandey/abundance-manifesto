# Translations

The Abundance Manifesto should be readable in every language.

## Available Translations

*None yet — be the first.*

## How to Translate

1. Fork this repository
2. Copy `MANIFESTO.md` to `translations/MANIFESTO_[language-code].md`
   - Use [ISO 639-1 codes](https://en.wikipedia.org/wiki/List_of_ISO_639-1_codes): `es` for Spanish, `fr` for French, `de` for German, `hi` for Hindi, `zh` for Chinese, etc.
   - Example: `translations/MANIFESTO_es.md`
3. Translate the content. Preserve the structure and headings.
4. Add yourself to the top of the translated file as translator:
   ```
   *Translated by [Your Name], [Date]*
   ```
5. Open a Pull Request titled `Translation: [Language name]`

## Notes for Translators

- Translate for meaning, not word-for-word. What matters is that the ideas land in your language.
- If a concept doesn't have a clean equivalent in your language, explain it — don't force an awkward literal translation.
- If you're aware of local examples of abundance principles in your country or region, feel free to add a brief note in your translation.

## Become a Regional Ambassador

If you translate the manifesto and want to help grow the movement in your region, open an issue introducing yourself. Regional ambassadors help recruit signatories, share the manifesto in local communities, and surface relevant open alternatives for their context.
